Introduction:
·Central processing unit(CPUs) live in practically all devices you own, whether it's a smartwatch, a PC, or a thermostat. They are responsible for processing and executing instructions, act as the brains of the devices.
·Graphics processing unit(GPU) a specialized processor originally designed to accelerate graphics rendering.GPU can process many pieces of data at the same time, making them useful for machine learning, video editing, and gaming applications.
GPU may be integrated into the computer's CPU or presented as a discrete hardware unit.

Central Processing Unit (CPU) Graphics Processing Unit (GPU)

CPU

CPUs are worked by putting billions of microscopic transistors onto a single computer chip. They're really minute entrances that switch on or off, consequently conveying into all that you do with the device, be it watching videos or writing an email.

A CPU takes instructions from a program or application and performs a calculation. The process will breaks down into three key stages, fetch, decode, and execute.

The original CPU had only one processing core, it's only allow it to perform one instructions at once. Now, the new CPU consists of multiple cores that allow it to perform multiple instructions at once. Also means packing a few CPUs on a single chip

GPU

GPUs can use for gaming, video editing, content creation, and machine learning.

Graphics programmers can use GPU to create more interesting visual effects and realistic scenes. GPUs are able to rendering graphics in both 2D and 3D. With better graphics performance, games can be played at higher resolution, and at faster frame rates.

Video editors and graphic designers may use GPU faster and easier to render video and graphics in higher-definition formats by using the parallel processing.

GPUs come in two basic types, integrated and discrete. An integrated GPU is embedded alongside the CPU. A discrete GPU is come on its own separate card.
